Politique de confidentialité

Extension Chrome Capsensfaker (v2.2.0) — Dernière mise à jour : 3 mars 2026

1. Présentation

Capsensfaker est une extension Chrome interne développée par Capsens. Elle permet de générer et remplir automatiquement des données de test dans les formulaires web, afin de faciliter les activités de QA (assurance qualité) de l'équipe.

2. Données collectées

L'extension accède aux données suivantes :

  • Adresse e-mail et profil Google — utilisés uniquement pour l'authentification via Google Sign-In et pour vérifier l'appartenance au domaine @capsens.eu.
  • Clé API Gemini — saisie volontairement par l'utilisateur pour activer la fonctionnalité de génération de projets par IA.

3. Stockage des données

Toutes les données sont stockées localement sur l'appareil de l'utilisateur via l'API chrome.storage. Aucune donnée n'est envoyée ni stockée sur des serveurs appartenant à Capsens.

4. Partage des données avec des tiers

L'extension ne partage aucune donnée personnelle avec des tiers. Les seules communications réseau sont :

  • Google OAuth2 — pour authentifier l'utilisateur (domaine @capsens.eu requis).
  • API Google Gemini — les requêtes sont envoyées uniquement lorsque l'utilisateur déclenche manuellement la génération de projet par IA, en utilisant sa propre clé API.

5. Permissions de l'extension

  • activeTab — pour interagir avec l'onglet actif lors du scan et du remplissage des formulaires.
  • scripting — pour injecter le script de remplissage dans les pages web.
  • storage — pour sauvegarder localement les préférences et la clé API.
  • identity — pour l'authentification Google Sign-In.
  • host_permissions — pour communiquer avec l'API Gemini et l'API Google.

6. Restriction d'accès

L'extension est réservée aux collaborateurs disposant d'une adresse e-mail @capsens.eu. L'authentification Google vérifie cette condition avant de permettre l'utilisation.

7. Sécurité

Les données sont stockées dans l'espace de stockage isolé (sandboxé) de l'extension Chrome. La clé API Gemini est conservée localement et transmise uniquement aux serveurs Google lors des appels API initiés par l'utilisateur.

8. Suppression des données

L'utilisateur peut supprimer toutes les données stockées par l'extension en la désinstallant depuis Chrome, ou en effaçant les données de l'extension via chrome://extensions.
